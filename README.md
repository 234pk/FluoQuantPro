## English

FluoQuantPro is a professional-grade quantitative analysis software specifically designed for fluorescence microscopy imaging. It is more than just an image viewer; it is a rigorous analysis platform that ensures scientific integrity through advanced processing pipelines.

![FluoQuantPro Demo](FluoQuantPro.gif)

### ğŸš€ Key Features
- **Dual-Track Processing:** Complete separation of **Measurement Track** (raw 8/16-bit scientific data) and **Rendering Track** (visual display with GPU acceleration).
- **Automated Data Import:** Intelligent dimension detection, automatic Squeeze & Maximum Intensity Projection (MIP) for Z-stacks and multi-dimensional data.
- **Smart Analysis:** Interactive Magic Wand tool for precise ROI selection and one-click quantitative measurement.
- **GPU Acceleration:** High-performance image enhancement and rendering using OpenCL.

### ğŸ›  Quick Start
1. **New Project:** Specify a folder containing your images; the system will automatically scan and organize samples.
2. **Enhance Image:** Use "Auto" or percentile sliders to get a clear view instantly.
3. **Measure:** Use the **Magic Wand (W)** to select targets and press **Ctrl+M** to generate reports.
